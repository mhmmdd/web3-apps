# Install pnpm for Windows to manage node packages
```shell
$ npm install -g pnpm

# add alias pn 
$ alias pn='pnpm'
```

# Create Faucet React App
```shell
$ node -v
v18.15.0

$ npm -v
9.3.0

$ pnpm create react-app faucet --template typescript
#$ npx create-react-app faucet --template typescript

$ cd faucet
$ pn install
```
